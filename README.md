# 개발환경
1. IDE: IntelliJ IDEA Community
2. Spring Boot 3.4.1
3. JDK 11
4. mysql
5. Spring Data JPA
6. Thymeleaf

# 게시판 주요기능
1. 글쓰기(/board/save)
2. 글목록(/board/)
3. 글조회(/board/{id})
4. 글수정(/board/update/{id})
   - 상세화면에서 수정버튼 클릭
   - 서버에서 해당 게시글의 정보를 가지고 수정화면 출력
5. 글삭제(/board/delete/{id})
6. 페이징처리(/board/paging)
